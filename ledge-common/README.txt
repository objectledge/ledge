Welcome to ObjectLedge!

ObjectLedge is an application development toolkit based on PicoContainer technology.
It can be used for developing applications working in command line, daemon and J2EE
environments.

ObjectLedge is OSI Certified Open Source Software. OSI Certified is a certification 
mark of the Open Source Initiative. See accompanying LICENSE.txt file for terms and 
conditions of distribution and use.
